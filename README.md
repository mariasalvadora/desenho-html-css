## Sobre
Brincando de desenhar com **CSS** e **HTML**.

---

## Como baixar

``` bash
# Clonar repositório
$ git clone https://github.com/mariasalvadora/desenho-html-css.git

# Entrar no diretório
$ cd desenho-html-css

# Instalar
$ liveserver 

# Abrir no browser
$ botão direito(index.html) - Open with live server

```
